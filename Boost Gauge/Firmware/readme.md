# Boost Controller Firmware (v0.2)

This folder contains firmware code for the **Boost Controller**.

> **v0.2 is the current best working version.**  
> Older versions are kept for version history and easy backtracking if needed.

## Folder layout

- `Libraries/` — Shared libraries used by the project. Heavily modified for flash space. (MAY NOT WORK IN OTHER PROJECTS)
- `BoostController_v0.2/` — **v0.2 (current / recommended)**

## Recommended use

- Start from **v0.2** for development, builds, and reference.
- Use older versions only for version history, comparison, or fallback if needed.

## Contributing

If you’re fixing or improving things:
- Please target **v0.2** as the main development version.
- Keep older versions unchanged unless there’s a specific reason to patch them.
