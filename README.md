# OpenTG

Repository for the custom automotive gauge projects of **OpenTG** - originally engineered for the **Hyundai Coupe/Tiburon**, but designed to be adaptable to a wide range of ECUs and setups.

**OpenTG AFR** is the first project under the OpenTG name, with more modules planned.

---

## Projects

- **OpenTG AFR Meter** (Wideband AFR/Lambda system)
  - Mainboard firmware + hardware (CJ125-based wideband controller, CAN, analog emulation)
  - Gauge firmware + hardware (TFT display, LEDs, menu, EEPROM settings)
